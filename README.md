## Installation

```sh
npm install -g lets-go
```

> Some MacOS and Linux users may need to use `sudo` with the above command.

## Supported Generations

### Templates

- React
- Libraries
- Express
- CLI

### Plugins

- Craco
- Typescript
- Tailwind
- Prettier
